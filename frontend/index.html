Automação Web

Projeto de automação web utilizando Python e Selenium para coletar dados de imóveis em sites de anúncios. O sistema automatiza a busca, extração e organização de informações relevantes para análise do mercado imobiliário.

Editor de Imagens

Ferramenta de automação para processamento em lote de imagens, desenvolvida em Python. O sistema permite dividir, redimensionar e otimizar múltiplas imagens simultaneamente, economizando tempo em tarefas repetitivas.

Análise de Dados

Sistema de análise de dados desenvolvido em Python para monitoramento de resultados da Mega-Sena. A aplicação coleta, processa e visualiza dados estatísticos, identificando padrões e tendências nos sorteios.

Calculadora de Horas

Ferramenta desenvolvida em Python para cálculo automático de horas de cursos. O sistema processa planilhas e documentos, organizando as informações por ordem alfabética ou numérica, facilitando o controle de carga horária.

API REST

API REST desenvolvida com Python e FastAPI para gerenciamento de dados. A aplicação oferece endpoints seguros e bem documentados, permitindo integração fácil com outros sistemas e aplicações web.

Site Responsivo

Site responsivo desenvolvido com HTML5, CSS3 e JavaScript. Design moderno e adaptativo que proporciona uma excelente experiência do usuário em qualquer dispositivo, desde smartphones até desktops.
